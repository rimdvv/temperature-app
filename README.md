## 추후 추가될 기능 

1. 대시보드를 만들어서 입력한 체온, 해열제 데이터를 line chart로 만들어 보여주는 기능 
2. 채팅에서 비디오 첨부 기능
3. 회원가입시 아이의 생년월일, 몸무게를 등록하여 해열제 추천 용량 계산해서 알려주는 기능
